# Vendor Payment Reconciliation System

## Overview
Invoice and payment matching

## Technology Stack (2013)
- **Tools**: MSSQL, SSIS

## Features

- Sample data and test scripts
- Complete documentation

3. Load database/seed-data.sql (if exists)
4. Configure application connection strings
